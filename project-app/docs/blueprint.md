# **App Name**: AfriVoice Collect

## Core Features:

- Prompt Display: Display prompts in English and local African languages for users to read.
- Speech Recording: Record user speech as they read prompts or provide spontaneous speech, such as describing a picture.
- Consent Management: Present a built-in consent form, age verification, and explanation of the study's purpose.
- Data Usage Options: Allow users to choose how their data can be used: research only, open dataset, or model training without release.
- Data Anonymization: Anonymize user metadata, including age range and region, while excluding personally identifiable information like names.
- Transcription: AI tool to automatically transcribe recorded speech.

## Style Guidelines:

- Primary color: Deep, earthy orange (#C46B13), reminiscent of the African continent, conveying warmth and authenticity.
- Background color: Light, desaturated beige (#F2E8D5), offering a neutral backdrop that ensures readability and reduces visual fatigue.
- Accent color: A muted, analogous reddish-brown (#9C4A0D) will provide visual contrast without being jarring.
- Font: 'PT Sans', a humanist sans-serif font, will be used for both headings and body text to create a modern, clean, and readable interface. This provides a balance of warmth and clarity suitable for diverse languages and user demographics.
- Use clear, universally understandable icons. Incorporate subtle cultural motifs where appropriate.
- Design a clean, intuitive layout, optimizing for mobile responsiveness.
- Use subtle animations for feedback on user interactions, such as button presses or form submissions.
